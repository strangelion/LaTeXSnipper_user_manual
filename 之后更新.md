## 方式一：直接执行（可能遇到瞬时故障）
```cmd
$env:Path = [System.Environment]::GetEnvironmentVariable("Path","Machine") + ";" + [System.Environment]::GetEnvironmentVariable("Path","User")
npx wrangler kv key put "pages/user_manual.html" --binding LATEXSNIPPER_KV --path ./user_manual.html --remote
```

## 方式二：使用带重试的脚本（推荐，自动处理 7009 等瞬时错误）
```powershell
.\deploy_kv.ps1
```
脚本会自动重试最多 5 次，遇到 `7009 Upstream service unavailable` 会等待后自动重试。
