截取识别

一键截图并自动识别 LaTeX 公式，支持手写与图像。

按下截图快捷键框选公式区域，内置 MathCraft ONNX 模型在本地完成 OCR 识别，无需上传任何数据到云端。支持印刷公式、手写体、图片内嵌公式等多场景识别，识别结果自动进入编辑区可供修改。

实时预览

编辑区支持实时渲染，Typst/MathJax/LaTeX 多种引擎。

编辑区支持 Typst、MathJax、LaTeX 三种渲染引擎实时预览，所见即所得。切换引擎无需重启，公式渲染效果即时呈现，方便在不同输出场景下对比排版效果。

导出与转换

支持 Pandoc 导出、Typst 转换以及多种图片格式。

支持 Pandoc 导出为 Word/HTML/Markdown，Typst 源码转换，以及 PNG/SVG/PDF 等多种图片格式导出。复制 LaTeX/MathML 可直接粘贴到 Word、Typora 等编辑器中。

本地模型

内置 MathCraft ONNX 引擎，离线可用，隐私安全。

内置 MathCraft ONNX 本地 OCR 模型，无需联网即可完成高精度公式识别。模型自动下载部署，首次启动通过依赖向导完成配置，后续离线使用零延迟。

外部模型

支持接入 Mathpix、SimpleTex 等外部 API 引擎。

除本地模型外，支持接入外部 API 模型（如 Mathpix、SimpleTex 等）获得更高识别精度。在设置中配置 API Key 即可切换，满足不同使用场景。

多平台

全面支持 Windows / Linux / macOS，体验一致。

全面支持 Windows、Linux、macOS 三大桌面平台。Windows 上可注册为自启动服务，Linux 下支持 Wayland/X11 截图，macOS 拥有原生风格界面。
